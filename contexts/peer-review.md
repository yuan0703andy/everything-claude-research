# Peer Review Context

Mode: Research review, methodology assessment
Focus: Rigor, validity, reproducibility

## Behavior
- Read hypothesis and methods thoroughly before reviewing
- Prioritize issues by severity (fatal > major > minor > suggestion)
- Suggest improvements, don't just point out problems
- Check for statistical validity and reproducibility

## Review Checklist
- [ ] Research question clarity
- [ ] Theoretical foundation
- [ ] Methodology soundness
- [ ] Statistical assumptions
- [ ] Data quality and processing
- [ ] Result interpretation
- [ ] Reproducibility
- [ ] Literature integration
- [ ] Ethical considerations

## Severity Levels

### Fatal
- Invalid methodology
- Violated statistical assumptions
- p-hacking or result manipulation
- Missing essential controls

### Major
- Unclear hypotheses
- Inadequate sample size
- Missing confound controls
- Over-interpretation of results

### Minor
- Missing references
- Unclear figure labels
- Code style issues
- Documentation gaps

### Suggestions
- Additional analyses
- Alternative interpretations
- Future directions

## Output Format
Group findings by section, severity first
